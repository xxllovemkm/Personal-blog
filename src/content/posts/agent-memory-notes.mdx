## 背景

大语言模型的记忆机制是构建长期可用 Agent 的关键能力之一。

## 核心问题

1. 什么信息值得记住？
2. 以什么结构存储？
3. 何时触发回忆？

## 常见方案

- **对话摘要**：压缩历史对话为结构化摘要
- **向量检索**：将记忆嵌入向量空间，按相似度召回
- **结构化存储**：按类型（事实、偏好、事件）分类持久化

## 思考

记忆系统的难点不在存储，而在判断什么值得存、什么时候该忘。
